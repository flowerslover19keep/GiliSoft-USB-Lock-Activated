# GiliSoft USB Lock

## Overview

GiliSoft USB Lock is a software solution designed to manage and restrict USB device access on Windows systems. Its primary purpose is to enhance data security by controlling the use of removable storage devices such as USB drives, external hard disks, and other portable media.

## Key Features

**Control USB device access with customizable permissions**  
**Block unauthorized USB storage devices effectively**  
**Monitor and log USB device activity for security tracking**  
**Support for various removable media including CD/DVD drives**  
**User-friendly interface designed for straightforward management**  
**Option to password-protect the application settings**  
**Ability to whitelist trusted USB devices for seamless use**  
**Automatic enforcement of device restrictions upon system startup**
